Unit 28 - PyWPS LST point statsÂ¶

A second version of the WPS process will return LST statistics for given point in Germany (referenced in EPSG:3035). A new input parameter (point) is defined on 18-19. The output is defined as JSON, the class is changed from LiteralOutput to ComplexOutput, see line 9,25-26.

The pixel extraction is done by t.rast.what on line 53. Statistics is computed based on command output manually on lines 62-84. Resultant Python dictionary is converted to JSON structure using json Python package on line 86. The json package is imported on line 3.

Note that also class name (28), process identifier (30), title (31), abstract (36) and version (37) needs to be updated.

 1#!/usr/bin/env python3
 2
 3import json 
 4from subprocess import PIPE
 5from datetime import datetime
 6
 7from grass.pygrass.modules import Module
 8from grass.script import parse_key_val
 9from pywps import Process, LiteralInput, ComplexOutput, Format
10
11
12class ModisV2(Process):
13
14    def __init__(self):
15        inputs = list()
16        outputs = list()
17
18        inputs.append(LiteralInput('coordinates', 'ETRS-89 coordinates',
19                                  data_type='string'))
20        inputs.append(LiteralInput('start', 'Start date (eg. 2023-03-01)',
21                                   data_type='string'))
22        inputs.append(LiteralInput('end', 'End date (eg. 2023-03-01)',
23                                   data_type='string'))
24
25        outputs.append(ComplexOutput('output', 'Computed LST statistics',
26                                     supported_formats=[Format('application/json')]))
27
28        super(ModisV2, self).__init__(
29            self._handler,
30            identifier="modis-v2",
31            title="Modis process (v2)",
32            inputs=inputs,
33            outputs=outputs,
34            # here you could also specify the GRASS location, for example:
35            # grass_location="EPSG:5514",
36            abstract="Computes LST stats for given location and period (limited to Germany and 2023).",
37            version="0.2",
38            store_supported=True,
39            status_supported=True)
40
41    @staticmethod
42    def _handler(request, response):
43        def check_date(date_str):
44            d = datetime.strptime(date_str, '%Y-%m-%d')
45            if d.year != 2023:
46                raise Exception("Only year 2023 allowed")
47
48        check_date(request.inputs['start'][0].data)
49        check_date(request.inputs['end'][0].data)
50
51        x, y = request.inputs['coordinates'][0].data.split(',')
52
53        m = Module('t.rast.what',
54                   strds='modis_c@PERMANENT',
55                   coordinates=[x, y],
56                   separator=',',
57                   where="start_time > '{}' and start_time < '{}'".format(
58                   request.inputs["start"][0].data,
59                   request.inputs["end"][0].data),
60                   stdout_=PIPE)
61
62        stats = {
63            'min' : None,
64            'max' : None,
65            'mean' : None,
66        }
67        tsum = 0
68        count = 0
69        for line in m.outputs.stdout.splitlines():
70            items = line.split(',')
71            if items[-1] == '*': # no data
72                continue
73            val = float(items[-1])
74            if stats['min'] is None:
75                stats['min'] = stats['max'] = stats['mean'] = val
76            else:
77                if val < stats['min']:
78                    stats['min'] = val
79                if val > stats['max']:
80                    stats['max'] = val
81            tsum += val
82            count += 1
83        stats['mean'] = tsum / count
84        
85        response.outputs['output'].data = json.dumps(stats)
86
87        return response
88
89
90if __name__ == "__main__":
91    from pywps.app.Service import Service
92
93    processes = [ModisV2()]
94    application = Service(processes)

Sample process to download: modis_v2.py

Important

Do not forget to import process as done in Unit 27 and restart demo PyWPS server.

Example of execute request:

http://localhost:5000/wps?request=Execute&service=WPS&identifier=modis-v2&version=1.0.0&datainputs=start=2023-03-01;end=2023-04-01;coordinates=4351849,3120313

Possible response.

<wps:ComplexData mimeType="application/json" encoding="" schema="">
{"min": -3.54999999999995, "max": 15.33, "mean": 4.673076923076951}
</wps:ComplexData>

Task

Try to improve the process in order to validate coordinates. Only point located in Germany is allowed.
