Unit 22 - Spatio-temporal scripting¶

GRASS offers a Python API for space-time processing. The usage is presented in the script described below.

In the script below similar computation to Unit 11 - PyGRASS scripting and Unit 12 - Script User Interface will be performed. But instead of processing single Sentinel scene, a spatio-temporal dataset containing multiple Sentinel scenes will be processed.

UI¶

Let’s define input and output parameters of the script:

  • b4 - Name of input 4th band space time raster dataset (line 19)

  • b8 - Name of input 8th band space time raster dataset (line 23)

  • mask - Name of the input mask (region+clouds) space time raster dataset (line 27)

  • output - Name for output stats file (line 30)

  • basename - Basename for output raster maps (line 33)

  • threshold - Threshold for removing small areas (line 38)

Python Temporal API¶

Functions from GRASS GIS Temporal Framework:

  • initialization must be done by init function, see line 144

  • space time datasets are open on lines 146-148 by open_old_stds

  • raster maps registered in reference dataset (b4) are listed on line 152 by get_registered_maps

  • related raster maps in two other datasets (b8, cl) are searched on lines 155-158 by get_registered_maps with where parameter

 51def cleanup():

112def stats(output, date, fd):
113    fd.write('-' * 80)
114    fd.write('\n')
115    fd.write('NDVI class statistics ({0}: {1})'.format(output, date))
116    fd.write('\n')
117    fd.write('-' * 80)
118    fd.write('\n')
119    from subprocess import PIPE
120    ret = Module('v.report', map=output, option='area',
121                 stdout_=PIPE)
122    for line in ret.outputs.stdout.splitlines()[1:]: # skip first line (cat|label|area)
123        # parse line (eg. 1||2712850)
124        data = line.split('|')
125        cat = data[0]
126        area = float(data[-1])
127        fd.write('NDVI class {0}: {1:.1f} ha'.format(cat, area/1e4))
128        fd.write('\n')
129
130    # v.to.rast: use -c flag for updating statistics if exists
131    Module('v.rast.stats', flags='c', map=output, raster='ndvi',
132           column_prefix='ndvi', method=['minimum','maximum','average'])
133    
134    data = vector_db_select(output)
135    for vals in data['values'].values():
136        # unfortunately we need to cast values by float
137        fd.write('NDVI class {0}: {1:.4f} (min) {2:.4f} (max) {3:.4f} (mean)'.format(
138            vals[0], float(vals[2]), float(vals[3]), float(vals[4])))
139        fd.write('\n')
140        
141def main():
142    import grass.temporal as tgis
143
144    tgis.init()
145
146    sp4 = tgis.open_old_stds(options['b4'], 'raster')
147    sp8 = tgis.open_old_stds(options['b8'], 'raster')
148    msk = tgis.open_old_stds(options['mask'], 'raster')
149
150    idx = 1
151    fd = open(options['output'], 'w')
152    for item in sp4.get_registered_maps(columns='name,start_time'):
153        b4 = item[0]
154        date=item[1]
155        b8 = sp8.get_registered_maps(columns='name',
156                                     where="start_time = '{}'".format(date))[0][0]
157        ms = msk.get_registered_maps(columns='name',
158                                     where="start_time = '{}'".format(date))[0][0]
159        output = '{}_{}'.format(options['basename'], idx)
160        compute(b4, b8, ms, output)
161        stats(output, date, fd)
162        cleanup()
163        idx += 1
164
165    fd.close()
166    
167    return 0
